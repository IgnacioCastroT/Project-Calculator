# Calculator

Proyecto desarrollado como parte del curriculum de **The Odin Project**, enfocado en reforzar la lógica de programación con JavaScript y la manipulación del DOM.

## Descripción

Esta aplicación es una calculadora funcional que permite realizar operaciones matemáticas básicas directamente desde el navegador.

El objetivo principal del proyecto fue practicar la construcción de una interfaz interactiva y la implementación de lógica matemática usando JavaScript puro.

## Objetivos de aprendizaje

- Manipulación del DOM.
- Manejo de eventos (`addEventListener`).
- Separación de lógica y presentación.
- Uso de funciones para operaciones matemáticas.
- Control de estado de la aplicación.
- Buenas prácticas en organización de código.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript (Vanilla JS)

## Funcionalidades

- Suma
- Resta
- Multiplicación
- División
- Botón para limpiar pantalla
- Manejo de errores básicos (ej: división por 0)
- Interfaz interactiva
